Auto Size Grid

grid-template-rows: 100px auto;

Row 1 is only a 100px in height, the flex items in column 1 wrapped, but get cut off. We only see 1 1/2 of the flex items

Row 2 is set to "auto" so, the flex items wrap and the grid area stretches to fit the content inside.

grid-template-columns: 200px auto;

Column 1 is fixed at 200px in width so it's enought to handle

The first column will be fixed at 200px, and the second column will shrink and expand according to the webstie's width.
